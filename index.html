ui by moolego

moolego ui is a minimal and extensible, object-oriented javascript ui based on mootools framework.

The components are based on a common foundation class (Core), which allows you to write powerful, flexible and cross-browser web applications with its elegant, well documented API. ## Oo & Extensible MooLego UI try to keep the power, extensibility and the elegant code syntaxe provided by Mootools. ## Pure Javascript, 100% Skinnable MooLego UI uses neither image nor CSS related files but is 100% skinnable. You just need to link the js packed files. ## Browsers compatibility MooLego UI is compatible with Safari 3+, ~Internet Explorer 6, Internet Explorer 7+, Firefox 2+ (gecko), Opera 9+ and Chrome.
