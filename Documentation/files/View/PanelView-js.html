UI.PanelView

Creates a panelView Object that let you browse inner views(panels)

Extends

UI.View

Arguments

options

Options

Returns

Panelview object.

Example

new UI.PanelView({
    url             : 'data.php?id=42',
    width           : 260,
    height          : 400,
}).inject(this.content);
