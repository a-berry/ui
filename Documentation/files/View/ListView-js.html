UI.ListView

The UI.ListView class defines objects that manage the list.

Extends

UI.View

Require

UI.Canvas UI.Skin UI.Element UI.View UI.Scrollbar

Arguments

options

Options

itemType(string)
scroll(boolean) react on scrollwheel if set to true
data(array)

Returns

Listview object.

Example

var listview = new UI.ListView({
    url             : 'data.php?id=42',
    width           : 260,
    height          : 400,
    scroll          : true
}).inject(this.content);
